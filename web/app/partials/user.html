About chrDB

chrDB is an open resource for the academic, commercial and hobbyist community. Its primary function is to provide a simple tool to research genomic data. In its infancy it will focus on the human genome but in the future other speicies will be added as time allows. Data will be stored from NCBI, OMIM and other open sources. This project started out as a learning tool for myself to learn the AngularJS framework but now I think it's worth releasing to the community and spending a few bucks on an Amazon Elastic Cloud instance and a domain name. chrdb.com is a great domain name don't you think?
